# Transformers: Recent Research Insights

## Key Papers
1. **[Vision Transformers Need Registers](https://arxiv.org/abs/2309.16588)**
   - Identifies artifacts in ViT feature maps and proposes a solution using additional tokens.
   - Improves performance for supervised and self-supervised models.

2. **[Transformers are SSMs: Generalized Models and Efficient Algorithms Through Structured State Space Duality](https://arxiv.org/abs/2405.21060)**
   - Connects Transformers and state-space models (SSMs) via structured semiseparable matrices.
   - Introduces Mamba-2, a faster and competitive alternative to Transformers.

3. **[Transformers Can Do Arithmetic with the Right Embeddings](https://arxiv.org/abs/2405.17399)**
   - Addresses positional tracking in arithmetic tasks using specialized embeddings.
   - Achieves 99% accuracy on 100-digit addition problems.

4. **[Transformers meet Neural Algorithmic Reasoners](https://arxiv.org/abs/2406.09308)**
   - Combines Transformers with GNN-based neural algorithmic reasoners (NARs).
   - Enhances algorithmic reasoning capabilities in language models.

5. **[Grokked Transformers are Implicit Reasoners](https://arxiv.org/abs/2405.15071)**
   - Explores implicit reasoning in Transformers through grokking.
   - Highlights systematic generalization challenges and solutions.

## Summary
Transformers continue to evolve, addressing limitations in vision, arithmetic, and algorithmic reasoning. Hybrid architectures and specialized embeddings are key trends.
